# ✨ Overview

**Rare Wiings** is a complete full-stack E-Commerce platform developed using **PHP, MySQL, HTML, CSS, and JavaScript**.
The platform provides a seamless online shopping experience with powerful admin management features, secure authentication, dynamic product handling, and modern responsive design.

This project demonstrates practical real-world development skills with scalable architecture and database-driven functionality.

# ⚙️ Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/parthmuliya-in/rare-wiings.git
```

---

## 2️⃣ Move Project Folder

Move the project folder into:

```bash
htdocs
```

For XAMPP users

OR

```bash
www
```

For WAMP users

---

## 3️⃣ Setup Database

* Open phpMyAdmin
* Create a new database
* Import the SQL file provided inside the project

---

## 4️⃣ Configure Database

Update database credentials inside:

```bash
config.php
```

Example:

```php
$host = "localhost";
$user = "root";
$password = "";
$database = "rare_wiings";
```

---

## 5️⃣ Run Project

Start:

* Apache
* MySQL

Then open:

```bash
http://localhost/rare-wiings
```
